== Installation ==


= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `subscribe-for-mailchimp-ajax.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard


= Using FTP =

1. Upload `subscribe-for-mailchimp-ajax` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings -> subscribe for mailchimp ajax
4. Fill fields the Api key, List id and Email
5. Use shortcode [sfma_form] on page
